ASSETS DIRECTORY
================

This directory contains assets for the BillMint application.

Required Assets:
----------------

1. logo.png
   - Place your BillMint logo here
   - Recommended: 1024x1024px PNG with transparent background
   - This will be used for app icon and splash screen
   - If you don't have a logo, create a simple one or use a placeholder

2. fonts/SourceCodePro-Regular.ttf
   fonts/SourceCodePro-Bold.ttf
   fonts/SourceCodePro-SemiBold.ttf
   
   - Download Source Code Pro font family from Google Fonts
   - Visit: https://fonts.google.com/specimen/Source+Code+Pro
   - Download and extract the TTF files to the fonts/ subdirectory
   - The app uses these fonts throughout the UI

Download Instructions:
----------------------

For Source Code Pro fonts:
1. Visit https://fonts.google.com/specimen/Source+Code+Pro
2. Click "Download family"
3. Extract the zip file
4. Copy the following files to assets/fonts/:
   - SourceCodePro-Regular.ttf
   - SourceCodePro-Bold.ttf
   - SourceCodePro-SemiBold.ttf

For logo.png:
1. Create or obtain a logo image
2. Resize to 1024x1024px
3. Save as logo.png in the assets/ directory
4. Run the flutter_launcher_icons generator (documented in README)
